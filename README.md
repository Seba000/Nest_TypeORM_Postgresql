# Api
1. Clonar el repositorio
2. iniciar el proyecto
```
yarn install
```
3. Levantar la base de datos entorno local (importante tener docker instalado)
```
docker-compose up -d
```
4. Levantar el proyecto
``` 
yarn start:dev
```
